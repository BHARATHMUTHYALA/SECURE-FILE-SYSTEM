Decrypt Encrypted File

Decrypt encrypted files using your encryption key

How It Works

  1. Upload your encrypted file - This is typically a .enc file that you exported or downloaded
  2. Enter the encryption key - The key you used when encrypting the file (password/passphrase)
  3. Click 'Decrypt' - The application will decrypt the file using your key
  4. Download the decrypted file - The original unencrypted file will be ready for download
Tip: Encryption keys are case-sensitive. Make sure to enter the exact key you used.
